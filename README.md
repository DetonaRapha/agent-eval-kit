# agent-eval-kit
"Evaluation toolkit for LLM agents: golden datasets, LLM-as-judge, and automated quality scorecards. Treats eval as a first-class testing layer." Ferramenta standalone que recebe um agente ou qualquer função de LLM, roda avaliação offline contra um golden dataset com um juiz automático, e cospe um scorecard (faithfulness, relevância, etc)
